🚨 PWA 网络问题解决方案

🔍 问题诊断

现象："Safari浏览器打不开网页，因为服务器已停止响应"

原因：离开局域网后，PWA缓存的开发服务器地址无法访问

✅ 解决方案

您的视频数据是安全的，存储在本地IndexedDB中。只需要重新部署应用即可。

📋 立即解决步骤

步骤1：清理当前PWA缓存

1. 在iOS主屏幕删除应用图标

2. 设置 → Safari → 清除历史记录与网站数据

步骤2：部署到云端（推荐）

运行以下命令部署到Netlify：

npm run build
npx netlify-cli deploy --prod --dir dist

或者手动上传到：

步骤3：本地静态服务器（临时方案）

在项目目录运行：

cd dist
python -m http.server 8080

然后访问：http://localhost:8080

步骤4：重新添加PWA

1. 在Safari中打开新的部署地址

2. 点击分享按钮 → "添加到主屏幕"

3. 您的视频数据将自动恢复

🌐 网络环境说明

开发环境：仅限局域网访问（localhost:3000）

生产环境：任何网络都可访问（云端部署）

视频数据：存储在本地，不受网络影响

⚠️ 重要提醒

您的视频文件存储在浏览器的IndexedDB中，数据不会丢失。只是应用界面需要稳定的网络访问。

🎉 推荐解决方案

1. 使用 Netlify 免费部署（永久域名）

2. 从主屏幕启动PWA获得最佳体验

3. 视频数据会自动恢复，无需重新上传
